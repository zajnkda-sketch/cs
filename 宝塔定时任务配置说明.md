# 宝塔定时任务配置说明

## 脚本功能

`auto_price_update.php` 是一个自动价格更新脚本，每天定时运行后会：

1. 自动读取 WordPress 数据库配置（无需手动填写密码）
2. 检查今日是否已有价格数据（避免重复写入）
3. 抓取最新化肥价格并写入数据库
4. 记录运行日志到 `auto_price_update.log`

## 安装步骤

### 第一步：上传脚本

将 `auto_price_update.php` 上传到网站根目录：

```
/www/wwwroot/1112.chaxunwa.com/auto_price_update.php
```

### 第三步：设置宝塔定时任务

1. 登录宝塔面板
2. 点击左侧菜单 **计划任务**
3. 点击 **添加任务**，填写以下信息：

| 字段 | 填写内容 |
|------|---------|
| 任务类型 | Shell脚本 |
| 任务名称 | 化肥价格自动更新 |
| 执行周期 | 每天 |
| 执行时间 | 09:00 |
| 脚本内容 | `php /www/wwwroot/1112.chaxunwa.com/auto_price_update.php` |

4. 点击 **添加任务** 保存

## 当前运行模式说明

脚本目前使用**模拟价格模式**（基于上次价格 ±0.8% 随机波动），适合在没有外部数据源时保持价格数据的连续性和真实感。

如需对接真实价格数据源，请修改脚本中的 `fetch_from_web()` 函数。

## 对接真实数据源（可选）

如果您有化肥价格数据服务商（如卓创资讯、隆众资讯等），可以在 `fetch_from_web()` 函数中添加 API 调用：

```php
function fetch_from_web($types) {
    $prices = [];
    
    // 示例：对接第三方价格 API
    $response = http_get('https://api.your-provider.com/fertilizer-prices', [
        'Authorization: Bearer YOUR_API_KEY',
    ]);
    
    if ($response) {
        $data = json_decode($response, true);
        foreach ($data['items'] as $item) {
            $prices[$item['name']] = floatval($item['price']);
        }
    }
    
    return $prices;
}
```

## 常见问题

**Q：脚本报错"找不到 wp-config.php"**  
A：请确认脚本中的 `WP_ROOT` 路径与实际 WordPress 安装路径一致。

**Q：脚本报错"数据库连接失败"**  
A：检查 wp-config.php 中的数据库配置是否正确，或数据库服务是否正常运行。

**Q：价格数据没有更新**  
A：查看日志文件，确认是否输出"今日价格已更新，跳过执行"（说明今天已经运行过了）。

**Q：如何手动触发更新**  
A：在宝塔终端执行：`php /www/wwwroot/1112.chaxunwa.com/auto_price_update.php`
